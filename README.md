# OnlyUsedTesla API

> Requires **Node >= 20.10** and **npm >= 10**.

## Quick start
```bash
# macOS/Linux
nvm install 22
nvm use 22
node -v && npm -v  # verify

npm install
cp .env.example .env
npm run start:dev

# Swagger -> http://localhost:3003/docs
# Health: /health, /health/cosmos, /health/storage
```
